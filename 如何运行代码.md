# 智颜方正系统运行指南

本文档说明如何配置和运行“智颜方正”系统的后端和前端。

## 环境要求

- Python 3.8+
- Node.js & npm
- Mac OS (本项目已针对 Mac M1/M2/Intel 进行优化)

## 1. 后端运行 (Backend)

后端使用 Flask 框架，并集成了 Stable Diffusion 模型（需较多内存）。

### 1.1 安装依赖
确保已安装必要的 Python 库。如果尚未安装，请运行：
```bash
pip install flask flask-cors opencv-python numpy mediapipe diffusers transformers accelerate torch torchvision
```

### 1.2 启动后端
进入 `backend` 目录并运行 `app.py`：
```bash
cd backend
python app.py
```

**注意事项：**
- **首次运行**：会自动下载 Stable Diffusion 模型（约 2-5GB），请保持网络通畅（代码中已配置 `hf-mirror.com` 镜像加速）。
- **启动时间**：加载模型可能需要 1-3 分钟，请耐心等待直到控制台显示 `Starting Flask server for Face Processing...`。
- **端口**：后端默认运行在 `http://0.0.0.0:5001`。
- **CPU 运行**：本项目配置为在 CPU 上运行推理，生成一张图片可能需要 30-60 秒，期间电脑可能会变慢，这是正常现象。

## 2. 前端运行 (Frontend)

前端使用 Vue 3 + Vite。

### 2.1 安装依赖
在项目根目录下运行：
```bash
npm install
```

### 2.2 启动前端
```bash
npm run dev
```
启动后终端会显示访问地址，通常为 `http://localhost:5173`。

## 3. 使用流程

1. 打开浏览器访问前端页面。
2. 在首页点击“开始体验”进入换妆页面（或先登录）。
3. **上传照片**：上传一张清晰的正脸照片。
4. **矫正**：系统会自动进行模糊检测和人脸矫正。
5. **选择风格**：点击“自然清透”、“轻熟职场”或“韩系爱豆”风格。
6. **生成**：系统将调用后端真实模型进行生成（此时后端终端会有进度条或日志显示），等待生成结果返回。
   - **注意**：由于放弃了模拟模式，改为真实调用，生成速度会比以前慢很多，请勿频繁点击。

## 4. 常见问题

- **报错 `IndentationError` 或其他语法错误**：请检查 Python 代码格式。
- **模型下载慢**：确保 `hf-mirror.com` 环境变量已生效（代码中已默认设置）。
- **内存溢出 (OOM)**：尝试关闭其他占用内存的软件。后端已配置 `enable_attention_slicing` 以优化内存使用。

## 5. 文件结构说明

- `backend/app.py`: 后端主入口，处理 HTTP 请求。
- `backend/makeup_gan.py`: 核心美妆生成逻辑（封装了 Stable Diffusion）。
- `backend/database/images/`: 存放上传的原图、矫正后的图片和生成的结果。
