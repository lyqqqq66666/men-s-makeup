# 如何使用 Beekeeper Studio 连接数据库

由于本项目使用的是 **SQLite** 数据库，您不需要像 MySQL 那样输入主机名、用户名或密码，只需要指向数据库文件即可。

## 连接步骤

1. **打开 Beekeeper Studio**。
2. **选择连接类型**：在主界面点击左侧的 `+` 号或 `New Connection`，然后选择 **SQLite**。
3. **选择数据库文件**：
   - 在 `Database File` 设置项中，点击文件夹图标进行浏览。
   - 导航到项目根目录下的以下路径：
     `/Users/LYQ/Desktop/大三资料/数字图像处理/智颜方正—智能图像矫正与风格男妆生成系统/backend/database/app_data.db`
4. **测试连接**：点击 `Test` 按钮，确保显示连接成功。
5. **保存并连接**：点击 `Connect` 或 `Save`。

## 注意事项

- **文件路径**：请确保指向的是 `app_data.db` 文件。
- **环境要求**：数据库文件是在运行后端代码后自动生成的。如果文件夹内没有该文件，请先运行一次后端代码（如 `python backend/main.py` 或初始化脚本）。
- **锁机制**：SQLite 是单文件数据库，如果你的后端程序正在进行大量写入操作，Beekeeper 在读取时可能会提示忙碌（Busy），一般情况下可以正常同时访问。

## 常用查询
一旦连接成功，你可以看到 `uploads`, `debug_images`, `corrected_images` 等表。你可以运行以下 SQL 查看数据：
```sql
SELECT * FROM uploads ORDER BY created_at DESC;
```
