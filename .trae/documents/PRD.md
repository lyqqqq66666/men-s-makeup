## 1. 产品概述

### 1.1 产品定位

**颜选MenX——男士个性化试妆与美妆智选平台** 是一款专为 18-35 岁中国男性打造的 **AI 智能图像矫正与风格男妆生成系统**。系统通过人脸检测、PCA 季型色彩分析、虚拟试妆引擎和个性化推荐算法，解决男性化妆入门的核心难题——"不知道怎么选、不知道怎么画、不知道适不适合"。

### 1.2 目标用户

| 维度 | 描述 |
|------|------|
| **年龄** | 18-35 岁男性 |
| **场景** | 职场面试、约会社交、日常通勤、重要场合 |
| **痛点** | 不知如何选化妆品、不清楚化妆步骤、担心效果不自然 |
| **技术水平** | 普通用户，零化妆经验 |
| **消费能力** | 中等，愿意为形象管理付费 |

### 1.3 核心价值

> **"让每位男性都能轻松拥有得体妆容，AI 帮你选对色号、画好每一笔"**

- 🎯 **精准推荐**：基于 PCA 季型分析，推荐最适合用户的色系和产品
- 💄 **虚拟试妆**：实时预览妆容效果，降低试错成本
- 🛒 **一键购买**：试妆满意直接购买推荐产品，形成商业闭环
- 📚 **方案保存**：保存个人妆容方案，随时复用

## 7. 用户旅程

### 7.1 核心用户旅程

```
注册/登录
    │
    ▼
上传人像照片
    │
    ├──→ 智能矫正（自动检测模糊、矫正角度）
    │
    ├──→ PCA 季型分析（分析肤色/发色，判定季型）
    │         │
    │         ▼
    │    查看季型报告（专属色盘 + 风格建议）
    │
    ▼
进入试妆工作台
    │
    ├── 选择分步试妆（底妆→眉→眼→修容→唇）
    │     └── 每步选择具体产品/SKU
    │
    ├── 或选择一键全妆（自然清透/轻熟职场/韩系潮流）
    │
    ├── 查看效果 → 撤销/重置/对比
    │
    ├── 保存妆容方案
    │
    └── 一键购齐推荐产品
              │
              ▼
         商城 → 购物车 → 结算
```

### 7.2 关键交互说明

**试妆工作台**（系统核心页面）：
- **左侧**：原图/效果图对比 + 5步试妆切换
- **右侧**：季型推荐标签 + 商品列表 + 套装推荐
- **操作**：点击商品即可实时预览上妆效果
