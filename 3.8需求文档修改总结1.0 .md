1. 目标

提升用户对“AI诊断—推荐—试妆—购买”流程的理解效率

提升试妆页的沉浸感与引导感

提升推荐到购买的转化闭环

2. 关键问题

首屏价值感不强，用户不易理解产品定位

PCA结果页偏展示，缺少解释和下一步引导

商城页个性化推荐感不明显

试妆页步骤感和默认推荐不足

保存方案、个人中心与购买链路衔接不够

loading / empty / error / retry 状态不完整

3. 重点优化项

首屏：增加“上传照片 → AI分析 → 推荐 → 试妆 → 购买”流程引导

PCA结果页：补充推荐解释、推荐风格、推荐商品入口

商城页：升级为个性化推荐商城，突出筛选态和推荐标签

试妆页：强化 Step 进度、推荐提示、默认组合、一键渲染

方案页 / 个人中心：增强保存、复用和一键购齐能力

全链路：补齐异常状态和重试机制

4. 优先级

P0
