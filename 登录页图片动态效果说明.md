# 登录页图片动态效果实现说明

本文档详细说明了 `src/views/Login.vue` 中登录页面的主要动态视觉效果及其实现原理。

## 1. 3D 旋转图片画廊 (Carousel)

登录页核心的视觉焦点是一个自动旋转的 3D 图片画廊。

### 实现原理
该效果完全基于 **CSS 3D Transforms** 实现，无需引入额外的 3D 库（如 Three.js）。

**关键技术点：**

1.  **3D 舞台构建**:
    *   容器 `.carousel-stage` 设置了 `perspective: 1000px`，为子元素创造 3D 景深感。
    *   旋转容器 `.carousel-container` 设置了 `transform-style: preserve-3d`，确保其子元素（图片）保留 3D 空间属性。

2.  **空间布局 (Cylinder Layout)**:
    *   共有 8 张图片 (`.carousel-item`)。
    *   每张图片通过 CSS 变量 `--i` (0-7) 标记索引。
    *   **旋转 (`rotateY`)**: 每张图片沿 Y 轴旋转 `45deg * i` (360 / 8 = 45度)。
    *   **位移 (`translateZ`)**: 旋转后，将每张图片沿 Z 轴向外推 `320px`。这使得图片排列成一个圆柱体表面。
    *   代码片段：
        ```css
        .carousel-item {
          transform: rotateY(calc(var(--i) * 45deg)) translateZ(320px);
        }
        ```

3.  **自动旋转动画**:
    *   `.carousel-container` 应用了无限循环的旋转动画 `spin`。
    *   动画从 `rotateY(0deg)` 旋转到 `rotateY(360deg)`，周期为 20秒。
    *   鼠标悬停时设置 `animation-play-state: paused` 实现暂停效果。

4.  **倒影效果**:
    *   使用了 `-webkit-box-reflect` 属性在图片下方创建渐变倒影，增强立体感。

## 2. 动态气泡背景 (Floating Bubbles)

背景中不断升起的半透明气泡效果是由 **TypeScript (JS) + CSS 动画** 结合实现的。

### 实现原理

1.  **JavaScript 动态生成**:
    *   组件挂载 (`onMounted`) 后，使用 `setInterval` 定时调用 `createBubble()` 函数。
    *   每次调用创建一个 `div.bubble` 元素并添加到 DOM 中。
    *   **随机化属性**:
        *   **大小**: 20px 到 80px 之间随机。
        *   **位置**: 水平位置 (`left`) 0% 到 100% 随机。
        *   **持续时间**: 动画时长 15s 到 25s 随机，使运动节奏自然不单一。
        *   **延迟**: 随机延迟出发。

2.  **CSS 复合动画**:
    *   气泡应用了两个并发动画：
        1.  **`floatUp` (线性)**: 控制气泡从底部 (`translateY(0)`) 飘升至顶部视口之外 (`translateY(-120vh)`)，并配合透明度变化（淡入淡出）。
        2.  **`wobble` (缓动)**: 控制气泡左右摇摆，模拟空气阻力下的真实漂浮感。

3.  **性能优化**:
    *   气泡动画结束后（`duration + delay`），JS 会自动移除该 DOM 节点，防止内存泄漏和页面卡顿。

## 3. 交互式指纹/手指引导

右侧的“探索一下”手指图标使用了 CSS 关键帧动画。

*   **动画**: 定义了 `bounceRight` 关键帧，使图标在 X 轴上进行 `translateX` 位移，产生持续的“指引”点击效果。

## 4. 登录框滑入效果

当点击探索按钮时，登录框会从右侧滑入。

*   **实现**: CSS Transition。
*   默认状态下登录框 `transform: translateX(100%)` (移出屏幕)。
*   激活状态 (`.active`) 设置为 `transform: translateX(0)`，配合 `cubic-bezier` 缓动函数，实现平滑且有质感的滑入效果。
